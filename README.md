# College Student Management System

 Project Description

This application was developed using Zoho Creator (Low-Code Platform).
It is used to manage student details, faculty information, attendance, and marks efficiently.

 Features

* Add and manage student records
* Manage faculty details
* Record student attendance
* Store and view marks
* Generate reports

 Technologies Used

* Zoho Creator
* Deluge Scripting
* Low-Code Development
